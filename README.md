# SpringBoot_Demo

### Used frameworks 

Java: 17
Spring Boot 3.2.0

### Quick Start 

**Before** development, please install the following service software:
- Maven
- PostgreSQL 

**Clone SpringBoot_Demo**

```
git clone https://github.com/geekidea/spring-boot-plus.git
cd SpringBoot_Demo
```

**Maven Build**

```mvn clean package```
